## Overview

The **Ex-Vivo Bladder Analysis Workflow** is a collection of MATLAB functions and scripts for analyzing physiological data obtained from the **Ex-Vivo Bladder Preparation**.

The workflow provides tools for:

- Categorizing experiments into user-defined experimental groups
- Extracting pressure-volume relationships
- Extracting pressure-afferent nerve relationships
- Extracting volume-afferent nerve relationships
- Analyzing transient pressure events (TPEs)
- Generating publication-quality plots of raw and processed data

---

## Requirements

- MATLAB R2024a or newer
- Signal Processing Toolbox

---

## Input Data

The workflow is designed for **Spike2 "Spreadsheet Text"** exports generated by **Cambridge Electronic Design (CED) Spike2**.

---

## Output

The import workflow produces a curated MATLAB (`*.mat`) data structure that serves as the canonical dataset for all downstream analyses.

---

# Main Workflow

### 1. ImportAndCurate_Spike2Data

Creates or appends the canonical curated MATLAB dataset.

Use this script to:

- Start a new dataset
- Append experiments to an existing dataset

↓

### 2. binningAnalysisGUI

Grouped 2D binning and visualization of:

- Pressure vs. Volume
- Pressure vs. Afferent Nerve Activity
- Volume vs. Afferent Nerve Activity

↓

### 3. TPEanalysisGUI

Transient Pressure Event (TPE) detection, quality control, and analysis.

---

# Installation

1. Download or clone this repository.
2. Place all source files in a single folder.
3. Add the folder to the MATLAB path.
4. Launch MATLAB.
5. Run:

```matlab
ImportAndCurate_Spike2Data
```

# Source Files

The repository contains the following MATLAB source files.

### assignExperimentalGroupsUI.m
**Version 1.0**

User interface for mapping detected condition keys to user-defined experimental groups.

---

### baseline.m
**Version 1.0**

Baseline correction using asymmetric least-squares smoothing.

---

### bin2DByGroup.m
**Version 1.0**

General grouped 2D binning for paired X/Y data vectors.

---

### bin2DByGroup_equalN.m
**Version 1.0**

Grouped trajectory binning using equal-N sequential bins instead of fixed-width bins.

---

### binningAnalysisGUI.m
**Version 1.0**

Graphical interface for grouped 2D binning functions.

---

### detectTPEs.m
**Version 1.0**

Detects transient pressure events (TPEs) using MATLAB `findpeaks()` on baseline-subtracted pressure signals.

---

### getConditionDictionary.m
**Version 1.0**

Returns the condition dictionary (default and user-defined) used for assigning experiments to groups.

---

### ImportAndCurate_Spike2Data.m
**Version 1.1**

Launches the data analysis workflow.

Builds or appends the canonical afferent nerve dataset from Spike2 Spreadsheet Text exports.

---

### inspectSpike2Txt.m

Read-only inspector for Spike2 Spreadsheet Text exports.

---

### loadSpike2Batch.m
**Version 1.0**

Loads Spike2 Spreadsheet Text files, selects channels, computes volume, and extracts filename metadata including:

- underscore tags
- Fill number
- experimental conditions
- recording windows

---

### plotBinned2DByGroup.m
**Version 1.0**

Plots grouped 2D binned data with X and Y error bars.

---

### plotRawPrepOverlaySparklines.m
**Version 1.0**

Plots raw-data sparklines with treatment overlays for each preparation.

---

### processSpike2Batch.m
**Version 1.0**

Processes pressure and nerve recordings by performing:

- smoothing
- baseline fitting
- peak-envelope fitting
- maximum volume calculation
- optional volume normalization

---

### TPEanalysisGUI.m
**Version 1.0**

Graphical interface for transient pressure event (TPE) analysis.
